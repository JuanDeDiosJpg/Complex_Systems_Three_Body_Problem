# Complex_Systems_Three_Body_Problem
Repository with the code used and animations 

In order to simulate the planetary orbits use file: "Tres_cuerpos_Cerradas_v3_energias_Planetas". Every Constant is already given and there are initial conditions for Earth, mars, Sun and Earth, Moon, Sun. Time step doesn't need to be changed

In order to simulate the non-planetary orbits use file: "Tres_cuerpos_Cerradas_v3_energias". There are as well the initial conditions for every orbit used and som more. The time step h needs to be changed depending on which orbit you need.

In order to graphicate orbits, animate them and see every equation used check "Graficar_y_ecuaciones.ipynb"

In order to check results about cr3bp and exponents check document "PCS.ipynb"
